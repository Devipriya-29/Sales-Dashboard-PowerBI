# 📊 Sales Dashboard (Power BI)

## 🔹 Project Overview
This project is a Sales Dashboard built using Power BI to analyze business performance.

## 🔹 Key Metrics
- Total Sales: 421K
- Total Profit: 72K
- Profit %: 17%
- Total Orders: 30

## 🔹 Insights
- West region has highest sales
- Electronics is the top category
- Laptop is the best-selling product
- UPI is the most used payment method

## 🛠 Tools Used
- Power BI
- DAX
